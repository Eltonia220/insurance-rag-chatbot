\# Insurance FAQ Chatbot \& Claims Prediction — RAG System



\## What this project does

Two AI/ML tools built for the insurance domain:

1\. A RAG-based chatbot that answers questions from insurance policy documents

2\. A machine learning model that predicts insurance claim likelihood with 92.93% accuracy



\## Tech Stack

\- Python, LangChain, ChromaDB, Google Gemini API

\- HuggingFace Sentence Transformers for embeddings

\- scikit-learn, pandas, RandomForestClassifier

\- RAG architecture for document-based question answering



\## Project 1 — Insurance FAQ Chatbot

Answers natural language questions from insurance PDFs using

Retrieval-Augmented Generation (RAG). Chunks documents, stores

them in ChromaDB, and queries Gemini to answer from context.



\## Project 2 — Claims Prediction Model

Trained on 58,000+ insurance records. Top predictive features

were subscription length, customer age, and vehicle age.



\## What I learned

\- RAG architecture and knowledge base construction

\- Prompt engineering strategies for LLM accuracy

\- Data preprocessing and ML model evaluation

\- Handling real-world imbalanced datasets
